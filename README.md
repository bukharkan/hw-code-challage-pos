Installation
------------

1. Clone a [https://github.com/ryanoasis/vim-devicons.git](https://github.com/ryanoasis/vim-devicons.git)
2. Install the Node Modules, 

    ```vim
    npm install
    ```

    > this for install all depedencies Node Modules

3. Running Server

    ```vim
    npm run start
    ```

	> this for run the api collection
